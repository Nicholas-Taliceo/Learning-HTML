About

Hello, world! My name is Nicholas Taliceo and this is my educational website about HTML and CSS!

The main goal of this website is to provide a resource for HTML and CSS as I teach myself the markup language. Most, if not all of these website will not be my own creations, but rather useful websites and tutorials that are already on the web. Perhaps, at some point, I will begin to include unique work to this website, but that will be far in the future.

What's more, this website has been created entirely in HTML/CSS by me alone. Of course, many of these aforementioned websites helped significantly in the creation of this website, but ultimately it was I who created it.
