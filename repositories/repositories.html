Repositories

  • Mozilla Developer Network -- This is an amazing source that has not only strong reference material but also very useful tutorials on HTML, CSS, JavaScript, and other web development software. This is probably my first choice when searching for HTML/CSS help.
  • The World Wide Web Consortium (W3C) -- This is the leading organization for web development and web standards. This website has a rich database of useful information.
  • W3 Schools -- This is a web developers site that is designed for tutorials and reference information about HTML, CSS, Javascript, and more! Update: I recently heard of the potential misguided advice that this website may/may not ocassionally portray. I still use it for some quick solutions, but I now seek other options.
  • HTML Goodies -- This is a great web resource for HTML and other web-related programming languages.
